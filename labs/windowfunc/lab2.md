## Задачи на LAG и LEAD

### Задача 1. Разница в зарплате с предыдущим сотрудником
Для каждого отдела (`department`) выведите:
- имя сотрудника
- зарплату
- зарплату предыдущего сотрудника (по возрастанию `id`)
- разницу между зарплатой текущего и предыдущего сотрудника

Для самого первого сотрудника в отделе в качестве предыдущей зарплаты укажите 0.

---

### Задача 2. Сравнение со следующим сотрудником
Для каждого отдела выведите:
- имя сотрудника
- город
- зарплату
- зарплату следующего сотрудника (по возрастанию `id`)
- текстовую метку: 
  - 'Выше' — если зарплата текущего сотрудника выше следующего
  - 'Ниже' — если ниже
  - 'Равна' — если равна
  - 'Последний' — если следующего сотрудника нет

---

### Задача 3. Смещение на 2 позиции
Для всех сотрудников выведите:
- имя
- отдел
- зарплату
- зарплату сотрудника, который был принят на 2 позиции раньше (по возрастанию `id` в рамках отдела)

Если такого сотрудника нет, выведите 'Нет данных'.

## Комбинированные задачи

### Задача 11. Динамика зарплат
Для каждого отдела выведите:
- имя сотрудника
- зарплату
- зарплату первого сотрудника отдела
- зарплату последнего сотрудника отдела
- разницу между последним и первым (абсолютный прирост зарплаты в отделе)

---

### Задача 12. Аномалии зарплат
Для каждого сотрудника определите, является ли его зарплата аномальной:
- 'Аномально высокая' — если зарплата выше, чем у следующих двух сотрудников (по `id`)
- 'Аномально низкая' — если зарплата ниже, чем у предыдущих двух сотрудников
- 'Норма' — во всех остальных случаях

Указание: используйте `LAG` и `LEAD` с разными смещениями.

---

### Задача 13. Иерархия зарплат
Для каждого сотрудника выведите:
- имя
- отдел
- зарплату
- позицию в отделе (1, 2, 3... по возрастанию `id`)
- зарплату сотрудника, который занимает позицию `позиция + 2` (если такой есть)
- имя сотрудника, который был первым в этом отделе

---

### Задача 14. Цепочка из трех
Для каждого сотрудника, у которого есть как минимум два следующих сотрудника в отделе, выведите:
- имя текущего сотрудника
- его зарплату
- среднюю зарплату трех сотрудников: текущего, следующего и через одного

Указание: используйте `LEAD` с разными смещениями.

---

### Задача 15. Первый и последний из того же города
Для каждого сотрудника выведите:
- имя
- город
- отдел
- имя первого сотрудника в отделе из того же города (по `id`)
- имя последнего сотрудника в отделе из того же города (по `id`)

Если в отделе нет других сотрудников из того же города, выведите NULL.

---

## Примечания для выполнения

1. Для `LAST_VALUE` и `NTH_VALUE` всегда явно указывайте фрейм:
   ```sql
   ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
   ```

2. Для `LAG` и `LEAD` фрейм не требуется.

3. Во всех задачах сортировка внутри партиции выполняется по полю `id` (как аналог порядка принятия на работу), если не указано иное.

4. Обратите внимание, что значения зарплат могут повторяться — это нормально для реальных данных.
